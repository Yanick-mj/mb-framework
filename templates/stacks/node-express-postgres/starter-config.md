# Node + Express + PostgreSQL — Starter Config

## Runtime
- Node.js 20+ LTS
- TypeScript strict mode
- ESM modules

## Framework
- Express.js
- Zod for validation
- pg or Prisma for DB access

## Database
- PostgreSQL 15+
- Migrations: dbmate, knex, or Prisma migrate

## Testing
- Jest for unit/integration
- Supertest for HTTP testing

## CI/CD
- GitHub Actions
- Lint + typecheck + test on PR
- Docker build on merge
